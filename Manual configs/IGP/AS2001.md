## R-2001-A

```c
conf t
!
ipv6 unicast-routing
!
router ospfv3 1
!
address-family ipv4 unicast
 redistribute rip metric 88
 router-id 2.2.4.1
 exit-address-family
!
address-family ipv6 unicast
 redistribute connected metric 5
 redistribute rip FILIAL metric 88 include-connected
 router-id 2.2.6.1
 exit-address-family
!
end
!
```

```c
conf t
!
interface f0/0
 ospfv3 1 ipv4 area 0
 ospfv3 1 ipv6 area 0
!
interface f0/1
 ospfv3 1 ipv4 area 0
 ospfv3 1 ipv6 area 0
!
end
!
```

```c
conf t
!
router rip
 version 2
 redistribute ospfv3 1 metric 5
 no auto-summary
 network 131.0.200.0
!
ipv6 unicast
 ipv6 router rip FILIAL
 redistribute connected metric 5
 redistribute ospf 1 metric 5 include-connected
!
interface f0/0
 ipv6 rip FILIAL enable
!
interface f1/0
 ipv6 rip FILIAL enable
!
end
!
```
