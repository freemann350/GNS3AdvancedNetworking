# AS 1001
## R-1001-A

```c
conf t
!
! DEFAULT ROUTES REQUIRED FOR IPv4 SINCE THERE IS NO IPv4 OSPF VIRTUAL LINK
!
ip route 0.0.0.0 0.0.0.0 130.0.0.2
ip route 0.0.0.0 0.0.0.0 130.0.0.6
!
ipv6 unicast-routing
!
router ospfv3 1
!
address-family ipv4 unicast
 redistribute bgp 1001
 router-id 1.1.4.1
 area 2 stub
 exit-address-family
!
address-family ipv6 unicast
 redistribute bgp 1001
 router-id 1.1.6.1
 area 2 stub
 exit-address-family
!
end
!
```
